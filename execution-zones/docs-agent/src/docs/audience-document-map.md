# 読者別ドキュメントマップ

**目的**: 開発者・意思決定者・ユーザー向けに、どの文書を用意するかを整理する。  
**対象読者**: ドキュメント担当者、プロダクトオーナー  
**最終更新日**: 2026-02-13

**フォルダ方針**: 対象読者でフォルダを分ける。文書ごとに 1 ファイル。パスは docs 直下からの相対パス。

```text
docs/
├── developer/        # 開発者向け
├── decision-maker/   # 意思決定者向け
└── user/             # ユーザー向け
```

---

## 1. 開発者向け（`developer/`）

| 文書                         | 内容                                                                 | 想定ファイル |
| ---------------------------- | -------------------------------------------------------------------- | ------------- |
| **クイックスタート**         | 初回デプロイ・環境構築の手順                                         | `developer/quickstart.md` |
| **アーキテクチャ**           | 概要・ゾーン構成・実装詳細・クロスアカウント                         | `developer/architecture.md` |
| **運用・デプロイ**           | 初回後の運用（Slack 設定、モニタリング・アラーム、再デプロイ）       | `developer/runbook.md` |
| **テスト**                   | テストの実行方法・範囲・検証観点                                     | `developer/testing.md` |
| **機能要件・設定リファレンス** | 機能要件一覧・設定項目（cdk.config 等）の参照                        | `developer/requirements.md` |
| **ADR**                      | アーキテクチャ決定と理由（一覧・索引）                               | `developer/adr.md` |
| **セキュリティ**             | 認証・認可、脅威モデル、要件、実装、CMK                               | `developer/security.md` |
| **トラブルシュート**         | 障害時の切り分け・対処・返信なし診断・フロー検証                     | `developer/troubleshooting.md` |

---

## 2. 意思決定者向け（`decision-maker/`）

| 文書                         | 内容                                                                 | 想定ファイル |
| ---------------------------- | -------------------------------------------------------------------- | ------------- |
| **企画書**                   | 背景・目的・インパクト・アーキテクチャ概要                           | `decision-maker/proposal.md` |
| **リスク・セキュリティ要約** | リスクと対策を 1 ページで。技術的裏付けは補足情報へ                  | `decision-maker/security-overview.md` |
| **補足情報**                 | 科学的・技術的裏付け（ナッジ、多層防御の根拠など）                  | `decision-maker/design-principles.md` |
| **コスト・リソースの目安**   | AWS 利用料の傾向、想定運用負荷。予算・採用判断用                     | `decision-maker/cost-and-resources.md` |
| **ガバナンス・利用ポリシー** | 誰が使うか、利用範囲、承認フロー（組織で必要な場合）                 | `decision-maker/governance.md` |

---

## 3. ユーザー向け（`user/`）

| 文書               | 内容                                                                 | 想定ファイル |
| ------------------ | -------------------------------------------------------------------- | ------------- |
| **使用方法**       | メンションの仕方、応答の待ち方、できること                           | `user/user-guide.md` |
| **注意事項・制約** | やってよいこと・悪いこと、機密情報、利用範囲                         | `user/usage-policy.md` |
| **FAQ**             | よくある質問（応答が遅い／返ってこない／添付ファイルの扱い等）       | `user/faq.md` |
